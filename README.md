# Generador de reporte mensual de primas pagadas
This is a simple yet effective tool designed to streamline the generation of policy reports. It automatically processes the monthly paid premiums report issued by the insurance company, transforming raw data into a clean and usable report.

The main goal is to reduce manual work and errors, enabling faster analysis of the policy portfolio and payment reconciliation.

### Key Features
Automation: Processes monthly report files (usually Excel) to extract key data from company reports

Reconciliation: Facilitates the verification of actually paid premiums against internal records.

Unified Report: Generates a standardized output file (e.g., a new CSV or Excel) ready for import or analysis.
